Название программы: Инженерный калькулятор
Версия: 2.0
Дата создания: 09 октября 2025

Описание:
Данная программа выполняет математические операции над введёнными числами.
Пользователь вводит числа через запятую, после чего программа вычисляет:
- сумму всех чисел
- среднее значение
- наибольшее и наименьшее число
- список квадратов чисел.

Данная программа разработана в рамках обучения NovaData по работе с Git и GitHub.

Пример работы:
Введите числа через запятую: 4, 7, 10, 3

Результат:
Сумма чисел: 24
Среднее значение: 6.00
Минимум: 3
Максимум: 10
Квадраты чисел: [16, 49, 100, 9]
